# Legacy finance import

This folder contains an extracted review CSV from the uploaded PDF export `finance - monthly.pdf`.

Important:
- The PDF is a visual export of one large spreadsheet page, not a real `.xlsx`/Google Sheet file.
- Because columns are compressed in the PDF, this extract is for review and mapping, not a safe automatic import.
- July 2026 is included in the PDF, but should not be imported over current app data. Use this CSV only up to June 2026 unless you intentionally reconcile July manually.
- For a reliable import into Supabase, export the original Google Sheet as `.xlsx` or CSV and upload that file.
